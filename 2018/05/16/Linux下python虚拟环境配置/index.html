Linux下python虚拟环境配置

Virtualenv

系统中的多个python混用会导致$PYTHONPATH混乱，或者各个工程对于package的版本要求不同等等情况。有一个简单的解决方案就是用virtualenv来隔离多个python，其本质只是实现隔离不同python中$PYTHONPATH的路径，当然也可以衍生到隔离多个$PATH。
