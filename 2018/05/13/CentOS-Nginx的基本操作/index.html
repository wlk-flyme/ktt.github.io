CentOS Nginx的基本操作

基本操作

解释
命令
安装
yum install nginx
启动
service nginx start
停止
service nginx stop
重载
service nginx reload

注意：

1:使用重载命令的前提：服务已经启动了，如果是 停止了，那么重载是失效的。

2:reload在运维中非常有用：使用restart命令时，启动会有一段时间。比如是0.5s，那么在这0.5s中用户就无法访问到修改的。相反，reload可以实现无缝连接，用户仍然可以访问到
